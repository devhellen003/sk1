What is a Defi Origin?

Defi Origin is an open source protocol for connecting decentralized applications to mobile wallets with QR code scanning or deep linking. A user can interact securely with any Dapp from their mobile phone, making Decentralized wallets a safer choice compared to desktop or browser extension wallets.
